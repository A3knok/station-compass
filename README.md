## 🚆サービス概要
――― 大型駅構内での迷子を解消するサービス ―――

駅降車後の「迷子状態」を解消する、改札から出口までに特化したナビゲーションサービスです。
駅構内の構造を基に、最適なルートを案内します。
観光客やビジネスマン、上京した方々が大型駅で迷うことなく、確実に目的の出口に辿り着けるサポートを提供します。
大型駅を日常的に使用しているユーザーは困っているユーザーのために自分が知っている駅構内のルート詳細を投稿することで迷子状態のユーザーの助けになります。

## サービス開発の背景
私は以前大型駅で迷い、目的の出口にたどり着けず冷や汗をかいた経験があります。 特に上京直後や出張中、観光中など「時間の余裕がない」場面では、駅構内での迷子は大きなストレスです。 
実際の駅構内図はエレベーターの位置や出口までのルートが大型駅であるほど分かりづらく、目的の出口までスムーズに移動することが難しいと感じました。

そこで、実際に駅を利用している人が「おすすめのルート」を投稿・共有し、困っている人はサービス内でルートを検索できる仕組みを作ることで、迷うことなく目的の出口へ到着できる体験を提供したいと考えました。

日常的に大型駅を利用する人がルートを投稿し、その情報が困っている人の助けとなり助けられた人がさらにルートを共有することで新たな利用者の助けにつながるサイクルが生まれます。

このようにユーザー同士で情報を共有し合うことで、駅構内で迷う人を少しずつ減らしていきたいという思いから本サービス「駅コンパス」を開発しました。
