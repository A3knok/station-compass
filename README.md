## サービス概要
駅降車後の「迷子状態」を解消する改札から出口への専門ナビゲーションサービスです。
駅構内の構造データを基に案します。
観光客やビジネスマン、上京した方々が大型駅で迷うことなく、
確実に目的の出口に辿り着けるサポートを提供します。

## このサービスへの思い・作りたい理由
就職で上京した際利用した大型駅構内で迷い遅刻しかけて冷や汗をかいた経験があります。
旅行での貴重な観光時間、大事な取引先との商談に向かうビジネスマン、上京直後で地理に詳しくない方々に時間にも心にも余裕をもって過ごしてもらうためこのサービスを利用していただきたいです。

## ユーザー層について
- メインターゲット：観光客、出張で慣れない駅を利用するビジネスパーソン、上京者
- サブターゲット：通勤・通学で大きな駅を利用する20-30代
- 利用シーン：急いでいる時、初めて利用する駅、混雑時の効率的な移動

ユーザー層は日常的に大型駅を利用するかしないかを大きな軸としました。
効率的な移動にも目を向けて通勤・通学で駅を利用する20-30代もサブターゲットとしています。
20-30代にしている理由は登校時間の幅が広い大学生(=駅利用時間も様々)と転職によって職場が変わる世代(=通勤ルート変更)と考えています。

## サービスの利用イメージ
### 観光客の場合
東京駅に到着した観光客が、八重洲口に向かいたい時、
改札の位置と目的出口を選ぶだけでルートが表示され、
貴重な観光時間を駅で無駄にすることなく目的地に到着できます。

### ビジネスマンの場合
出張先の渋谷駅でハチ公出口に向かう必要がある時、
複雑な駅構内でも確実なルート案内により、
大事な商談に遅刻するリスクを回避できます。

### 上京者の場合
東京生活を始めたばかりの方が、慣れない駅でも
安心して移動でき、東京の複雑な駅への不安が軽減されます。

## ユーザーの獲得について
- 自分の体験談をブログ・SNSで発信(note,X)
- プログラミング学習コミュニティでの紹介
- 口コミ促進機能の実装

## サービスの差別化ポイント・推しポイント
- 駅構内案内：改札から駅の出口までを案内。目的地までの残り100m問題を解消
- ユーザー参加型：コメントや評価による情報の質向上
- 視覚的な目印: 「○○の看板を右に曲がる」など、具体的なランドマーク情報
